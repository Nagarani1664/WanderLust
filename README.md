# WanderLust 🌍

WanderLust is a full-stack travel listing web application inspired by Airbnb. Users can explore travel destinations, create listings, upload property images, leave reviews, and manage their accounts securely.

## 🚀 Live Demo

[https://wanderlust-kisp.onrender.com](https://wanderlust-kisp.onrender.com)

---

# ✨ Features

* User Authentication (Signup/Login/Logout)
* Create, Edit, and Delete Listings
* Upload Images with Cloudinary
* Reviews and Ratings System
* Wishlist Feature
* Search Destinations
* Category-Based Filtering
* Geocoding with OpenStreetMap
* Session & Flash Message Handling
* Responsive UI using Bootstrap
* MongoDB Atlas Database Integration
* Fully Deployed on Render

# ⚙️ Installation

## 1. Clone Repository

```bash
git clone https://github.com/Nagarani1664/WanderLust.git
cd WanderLust
```

## 2. Install Dependencies

```bash
npm install
```

## 3. Create .env File

```env
ATLASURL=your_mongodb_connection_string
SECRET=your_secret_key

CLOUD_NAME=your_cloudinary_cloud_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_api_secret
```

## 4. Run Application

```bash
node app.js
```

or

```bash
nodemon app.js
```
